# 🌸 Factory 4flo

> **A local-first AI Software Factory that transforms ideas into structured engineering workflows.**

Factory 4flo is a single-file application that helps humans and AI collaborate throughout the entire software development lifecycle.

Instead of losing context in endless chat conversations, Factory 4flo organizes every project into a **Factory Passport**—a structured record of ideas, specifications, implementations, approvals, deployments, and project history.

Everything runs locally in your browser by default.

No installation.

No backend.

No database.

Just open one HTML file and start building.

---

# 🚀 5-Minute Quick Start

You can be productive in less than five minutes.

## 1. Download

Clone the repository or download the HTML file.

```bash
git clone https://github.com/MRReedii/factory_4flo.git
```

or simply open

```text
4flo.html
```

inside your favorite browser.

---

## 2. Start a Factory Run

Open **Factory Passport**.

Inside **Create Factory Run**, enter something like:

```text
Build a login page with GitHub authentication.
```

or

```text
#5368
#5366

Improve deployment pipeline.
```

Click:

**Start Factory Run**

Factory 4flo immediately creates a structured engineering workflow.

---

## 3. Review Your Snapshots

Each Factory Run becomes a series of snapshots.

```text
Context

↓

Specification

↓

Implementation

↓

Proof
```

Every snapshot records one stage of development.

---

## 4. Configure Plugins (Optional)

Open **Plugins**.

Choose plugins for:

* AI Models
* Git Providers
* Deployment Platforms
* Execution Tools

You can remain completely local or connect external services.

---

## 5. Export

When you're finished:

Click **Export JSON**

Your entire Factory Passport becomes portable.

You can:

* archive it
* share it
* import it later
* continue development anywhere

# 📖 What is Factory 4flo?

Factory 4flo is **not another AI chat interface.**

Think of it as a software engineering operating system.

Instead of generating one answer at a time, Factory 4flo manages an entire project from beginning to end.

It organizes:

* Ideas

* Human approvals
* Timeline history
* Git information
* Deployments
* Credentials
* Plugins
* Automations

Everything stays structured instead of disappearing into chat history.

---

# 🏭 Core Concepts

## Factory Passport

A Factory Passport represents one complete software project.

Each passport contains everything needed to understand, reproduce, or continue development later.

It includes:

* Project details
* Snapshots
* Timeline
* Plugin selections
* Credentials
* Integrations
* Deployment information
* Exportable JSON

---

## Snapshots

Snapshots are permanent checkpoints.

Instead of replacing previous work, Factory 4flo records every important milestone.

Typical workflow:

```text
Context

↓

Specification

↓

Implementation

↓

Proof
```

Snapshots can be:

* Approved
* Rejected
* Edited
* Shared
* Exported
* Replayed

---

## Timeline

Every meaningful action becomes an event.

Example:

```text
Run Created

↓

Specification Generated

↓

Human Review

↓

Approved

↓

Git Commit

↓

Deployment Started

↓

Deployment Complete
```

Nothing disappears.

---
